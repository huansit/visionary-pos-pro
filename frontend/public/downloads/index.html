Downloading the app alone does not grant access to your POS.
1. Download and run the Windows installer.
2. In the admin portal, generate a terminal activation code for the correct branch.
3. Open the desktop app and enter the activation code on the first-run activation screen.
4. After activation, cashiers sign in with their employee number and PIN.
